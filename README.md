# Stock-market-Dashboard-2022-24
Title:- 📊 Stock Market Dashboard (2022–2024)
An interactive Stock Market Dashboard built to analyze and visualize key financial metrics across multiple sectors for the years 2022–2024. This project provides insights into market trends, sector performance, and volatility using powerful data visualization techniques.

* Tools & Technologies Used
Microsoft Excel / Power BI (for dashboard creation)

Data Source
	Kaggle Dataset

Features
	KPI Summary Cards
	Sector-wise Analysis
	Volatility Tracking
	Volume Insights
